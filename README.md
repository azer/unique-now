## unique-now

Module to generate unique unix times.

```js
now = require('unique-now')

i = 5
while ( i --> 0 ) console.log( now() )
// => 1371752900312
// => 1371752900313
// => 1371752900314
// => 1371752900315
// => 1371752900316
```

## Install

```bash
$ npm install unique-now
```
